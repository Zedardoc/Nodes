## 📌 Descripción

Este proyecto implementa **estructuras de datos dinámicas** basadas en **listas enlazadas** y **listas doblemente enlazadas**.  
El objetivo es demostrar cómo funcionan internamente estas estructuras, cómo manipular nodos dinámicamente y cómo realizar operaciones comunes como inserción, eliminación y recorrido.

Las listas enlazadas permiten almacenar datos de manera **dinámica en memoria**, evitando la necesidad de estructuras de tamaño fijo.

---

# 📚 Contenido del Proyecto

El proyecto incluye implementaciones de:

- 🔹 **Lista Enlazada Simple**
- 🔹 **Lista Doblemente Enlazada**

Cada estructura incluye operaciones fundamentales.
